# Invite telegram bot

Telegram bot for referral management and USDT withdrawal system.

## Features

- 🌐 Multi-language support (Russian, English)
- 👥 Referral system with rewards
- 💸 USDT TRC20 wallet withdrawals
- 🛠️ Admin panel for management
- 📊 User statistics
